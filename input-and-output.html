MNet Manual

2 Input and output

2.1 Data input

The sample data comprises an integrated analysis of metabolomic and transcriptomic profiles in triple-negative breast cancer.

  • Metabolite Data: an interactive table for user-input metabolic data with rows corresponding to metabolites and columns corresponding to samples.

  • GeneExp Data: an interactive table for user-input metabolic data with rows corresponding to genes and columns correspond to the samples.

  • Group Data: Group information.

2.2 Result output

The analysis results can be output in image form in two formats: PDF and JPEG. The figure width, height and resolution(DPI) can be adjusted.
